# Brand Bible

This folder defines how VikashOS should sound, what it should stand for, and how the public narrative should stay consistent across LinkedIn, projects, website copy, resume material, and future products.

Use this folder before writing or publishing anything public.

## What Lives Here

- Brand positioning and messaging rules.
- Content DNA and writing style.
- The canonical [Content DNA Library](Content-DNA-Library/README.md) and selection matrix.
- Permanent content series.
- Voice, tone, and quality standards.
- Rules for what VikashOS should never become.

## Practical Use

Before drafting content, check:

- Does this connect to business growth, buyer psychology, AI, leadership, or continuous learning?
- Does it sound like a real professional speaking from experience?
- Is there a useful observation, framework, or takeaway?
- Is anything exaggerated, invented, or written only to attract attention?

If the answer feels weak, improve the idea before writing the post.
