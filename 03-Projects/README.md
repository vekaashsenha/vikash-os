# Projects

This folder tracks practical projects inside VikashOS.

Projects should connect learning with execution. Each project should have a clear problem, user, first version, learning log, and reuse plan.

## Project Principles

- Start small.
- Build for a real use case.
- Document decisions.
- Reuse what works.
- Share learning only after it is clear and useful.
