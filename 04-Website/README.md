# Website

This folder stores strategy, copy, structure, and future updates for the VikashOS website.

## Website Role

The website should explain Vikash's professional identity, learning journey, projects, content series, and business perspective in one clear place.

It should not feel like a generic portfolio. It should feel like the home base for an experienced marketing leader building with AI and documenting practical learning.

## Suggested Sections

- Professional identity.
- VikashOS narrative.
- Content series.
- Projects.
- Selected writing.
- Resume and profile links.
- Contact or collaboration intent.

## Copy Rule

Keep the language direct, credible, and human. Avoid inflated claims and vague phrases.
