# Knowledge Vault

The Knowledge Vault stores reusable thinking for VikashOS.

This is where raw observations, research notes, frameworks, stories, business ideas, and learning reflections are captured before they become posts, website copy, projects, or strategy documents.

## Capture Rule

Capture the note even if it is rough. Refine it later.

Every useful note should include:

- Source or context.
- Key observation.
- Why it matters.
- Possible use.
- Tags or related series.

## Folder Guide

| Folder | Use |
| --- | --- |
| `Marketing` | Marketing strategy, demand generation, brand, GTM, and revenue notes. |
| `AI` | AI tools, workflows, prompts, app ideas, and learning logs. |
| `Business` | Strategy, finance, operations, economics, and leadership concepts. |
| `Buyer-Psychology` | Industry-specific buying behaviour and stakeholder insights. |
| `AGMP` | IIMA AGMP learning notes and applied reflections. |
| `Frameworks` | Reusable models, checklists, and decision structures. |
| `Stories` | Real experiences that may become posts after review. |
| `Research` | Public research, reports, articles, and source-backed material. |
