# Frameworks

Use this folder for reusable models, decision checklists, content structures, marketing frameworks, buyer psychology maps, and operating systems.

Frameworks should simplify thinking without pretending that business problems are always simple.

## Starter Frameworks To Build

- Observation, Business Context, Lesson, Framework, Practical Takeaway, Question.
- Buyer committee mapping.
- Build once, use everywhere.
- Learn, build, share, repeat.
- Marketing beyond campaign execution.
